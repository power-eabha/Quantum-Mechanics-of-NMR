# Quantum-Mechanics-of-NMR
Numerical simulations supporting a study of spin dynamics, Bloch equations, and resonance behaviour in NMR systems.


## Purpose

The code was developed to:
- verify analytical results (e.g. Rabi oscillations and resonance behaviour)
- compare time-domain and frequency-domain approaches
- demonstrate equivalence between laboratory and rotating frame descriptions

It is intended as a supporting computational tool rather than a standalone project.

## Methods Implemented

- Time-dependent Bloch equation solver
- Rotating frame transformations
- Numerical extraction of resonance frequencies
- Frequency-domain analysis of magnetisation signals

## Key Results Supported

- Rabi oscillations and transition probabilities  
- Lab frame vs rotating frame equivalence  
- Dependence of resonance frequency on detuning  
- Agreement between analytical and numerical approaches
